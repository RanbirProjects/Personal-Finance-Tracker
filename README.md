✨ **Features**

• User authentication (Register / Login)  
• Add, edit, delete expenses  
• Expense categorization  
• Expense history tracking  
• Dashboard with statistics  
• Charts for expense analysis  
• Responsive UI

---

⚡ **Tech Stack**

React • TypeScript • Material UI • Recharts • Axios • React Router  
Node.js • Express.js • MongoDB • Mongoose • JWT • bcryptjs

---

🚀 **Run Locally**

```bash
git clone https://github.com/RanbirProjects/Personal-Finance-Tracker.git
cd Personal-Finance-Tracker
cd backend && npm install
cd ../frontend && npm install

Create .env in backend

PORT=5000
MONGODB_URI=mongodb://localhost:27017/expense-tracker
JWT_SECRET=your-secret-key

Start project

cd backend && npm run dev
cd frontend && npm start

Frontend → http://localhost:3000

Backend → http://localhost:5000
